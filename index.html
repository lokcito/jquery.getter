Mejor, Utiliza una abstracci칩n:

	'url' : 'url',
	'params' : 'param=12',
	'success' : function(res) {
		//hacer algo
	},
	'empty' : function(res) {
		//response vacio
	},
	'error' : function(err) {
		//hacer algo
	}

Jquery Getter, has the following methods:

  • success(response)
  • empty(response)
  • complete(response)
  • error(response)

Jquery Getter, has the following options:

  • type: [api, views, [undefined]]
  • url: [api[short url], views[show url], undefined[full url]]
  • params: &param=value&param=value
