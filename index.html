Your project works with Apis?

Jquery is an extension of jquery getter. With which you can better manage your calls to a service, api or resource.

It is necessary, install and configure. Then forget about a long path of service, api or resource you request through getter.

Here is a usage example: 
$.getter({
	'type' : 'api',
	'url' : 'api/customers',
	'params' : '&id=' + 1,
	'success' : function(response){
		some things ...
	},
	'error' : function(){

	}
})

Jquery Getter, has the following methods:

  • success(response)
  • empty(response)
  • complete(response)
  • error(response)

Jquery Getter, has the following options:

  • type: [api, views, [undefined]]
  • url: [api[short url], views[show url], undefined[full url]]
  • params: &param=value&param=value
